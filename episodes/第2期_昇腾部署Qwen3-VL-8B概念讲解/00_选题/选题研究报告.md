## 三、3个核心类比

**类比1：宜家家具套装 → Docker容器镜像**
- 镜像=宜家大礼包（木板、螺丝刀、图纸都装好了）
- docker pull=网上下单
- docker run=打开箱子组装

**类比2：厨房外卖间 → 容器隔离**
- 宿主机=主厨房
- 容器=独立外卖间，有自己的厨具
- 用完关闭，主厨房干干净净

**类比3：外接显卡坞 → 挂载NPU设备**
- NPU=外接显卡坞
- `--device /dev/davinci0` = 插上USB-C线
