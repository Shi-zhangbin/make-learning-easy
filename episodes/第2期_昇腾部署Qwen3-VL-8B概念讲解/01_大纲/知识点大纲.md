# 知识点大纲：昇腾部署Qwen3-VL-8B概念讲解

> 第2期 | 15分钟 | 纯小白向 | B站

---

## 知识体系总图

```
你有一台服务器（昇腾）
  └─ 装了一块 NPU（专用AI计算卡）
      └─ 想跑 Qwen3-VL-8B（能看图说话的AI模型）
          └─ 用 Docker 容器（隔离的"小厨房"）来运行
              └─ docker pull 下载镜像（像网购宜家套装）
              └─ docker run 启动容器（像拆箱组装）
                  └─ 容器通过 --device 挂载 NPU（像USB-C连显卡坞）
```

## 记忆链条
NPU ≠ GPU → Docker ≠ 虚拟机 → 镜像 ≠ 安装包 → docker pull = 收快递 → docker run = 开火做饭 → --device = 插线 → Qwen3-VL = 能看图的AI助手
