STEP 1 / 7

打开终端（Terminal）

🍎 Mac用户
Cmd+空格 → 输入"Terminal" → 回车
或者从 启动台 → 其他 → 终端
🪟 Windows用户
建议安装 Windows Terminal（微软商店免费）
或者用 PowerShell 也可以
终端窗口长这样 →
Last login: Mon Jun 2 09:00:00
shizhangbin@MacBook-Pro ~ % _
备选方案 · 不推荐

终端SSH连接（如果你没有VS Code）

基本命令
ssh 用户名@服务器IP地址 然后回车
# 假设用户名是 ubuntu，IP是 192.168.1.100
shizhangbin@MacBook-Pro ~ % ssh ubuntu@192.168.1.100

# 第一次连接会看到这个，输入 yes
The authenticity of host ... can't be established.
Are you sure you want to continue connecting? yes

# 输入密码（输的时候不显示，正常）
ubuntu@192.168.1.100's password: ********

# 连接成功！提示符变了
ubuntu@ascend-server:~$ _
💡 连不上？ ①IP对不对 ②用户名对不对 ③公司VPN连了没 ④服务器开机了没 ⑤问运维要开通权限
STEP 2 / 7 · ✅ 推荐方式

用VS Code Remote-SSH连接

① 安装Remote-SSH插件
VS Code左侧扩展栏 → 搜 "Remote - SSH" → 安装（微软官方，蓝色图标）
② 添加SSH目标
F1（Cmd+Shift+P）→ 输入 Remote-SSH: Connect to Host → 选 Add New SSH Host
输入：ssh ubuntu@192.168.1.100 → 回车 → 选择SSH配置文件保存
③ 连接！
点左下角绿色区域 → Connect to Host → 选你的服务器
输入密码 → 新窗口打开 → 你已经连上远程服务器了！
💡 连上后点"打开文件夹"就能看到服务器上的文件，像在本地一样编辑
VS Code连上后的样子 →
⚡ REMOTE SSH
┌──────────────────────────

STEP 4 / 7 · NVIDIA机器

看GPU状态：nvidia-smi

💡 在VS Code集成终端（Ctrl+`）里直接输入命令
一条命令看到所有GPU信息
输入 nvidia-smi 然后回车
ubuntu@gpu-server:~$ nvidia-smi

+-----------------------------------------------------------------------------+
| NVIDIA-SMI 545.23.08 Driver Version: 545.23.08 CUDA: 12.3 |
+-------------------------------+----------------------+----------------------+
| GPU Name Persistence-M| Bus-Id Disp.A | Volatile Uncorr. ECC |
| Fan Temp Perf Pwr:Usage/Cap| Memory-Usage | GPU-Util Compute M. |
+-------------------------------+----------------------+----------------------+
| 0 Tesla T4 Off | 00000000:00:1E.0 Off | Off |
| 59% 62°C P0 70W / 70W| 3242MiB / 15360MiB | 0% Default |
+-------------------------------+----------------------+----------------------+
🔍 怎么看
GPU 0 Tesla T4 → 显卡型号
70W / 70W → 功耗（满负载）
3242MiB / 15360MiB → 已用/总共显存
0% → GPU利用率（空闲）
📌 截图保存！这是你的第1个产出物
STEP 5 / 7 · 昇腾机器

看NPU状态：npu-smi info

昇腾也有类似的命令
输入 npu-smi info 查看NPU芯片状态
ubuntu@ascend-server:~$ npu-smi info

+--------------------------------------------------------------------------------------------+
| npu-smi 24.1.rc1 Version: 24.1.rc1 |
+-------------------------------+----------------------+----------------------+
| NPU Name | Health | Power |
| Fan Temp Temp(Tctl) | Bus-Id | Chip-Phys |
+-------------------------------+----------------------+----------------------+
| 0 310P3 | OK | 75.2W |
| N/A 55°C 61°C | 0000:B1:00.0 | 0 |
+-------------------------------+----------------------+----------------------+
🔍 怎么看
NPU 0 310P3 → 芯片型号
Health: OK → 芯片状态正常
75.2W → 当前功耗
55°C → 温度
📌 截图保存！第2个产出物
