STEP 3 / 7

连上后第一件事：看看周围

💡 VS Code连上后，按 Ctrl+`（反引号）打开集成终端，后面的命令都在这里执行
📂 我在哪？
pwd → 显示当前目录路径
📋 有什么文件？
ls -la → 列出所有文件和文件夹
👤 我是谁？
whoami → 显示当前用户名
ℹ️ 什么系统？
uname -a → 显示系统信息
ubuntu@ascend-server:~$ pwd
/home/ubuntu

ubuntu@ascend-server:~$ ls -la
total 48
drwxr-xr-x 7 ubuntu ubuntu 4096 ...
-rw-r--r-- 1 ubuntu ubuntu 220 ... .bash_logout

ubuntu@ascend-server:~$ whoami
ubuntu
