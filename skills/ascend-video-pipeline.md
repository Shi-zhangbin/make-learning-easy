# ascend-video-pipeline

输入主题 → 自动产出成品视频。

## 管线流程

T0(选题) → T1(大纲) → T2(口播稿) → T3(配音+字幕) → T4(分镜) → T5(配图) → T6(Composition) → T7(渲染)

管线位置：`~/Desktop/ascend-pipeline/`，Python 代码在 `v3/` 下。

## 内容生成步骤

以下步骤需要你直接（Codex）生成内容，不是靠 Python 脚本。

### T0：选题研究报告

当 engine 输出 `📝 需要生成选题研究报告` 时：

读取 `.step_prompt.json` 获取 topic，然后写入 `选题研究报告.md`，内容结构：

- 选题背景（为什么重要，适合谁）
- 核心知识点（5-8 个）
- 教学类比（2-3 个）
- 讲解路径
- 常见误区

### T1：知识点大纲

读取 `.step_prompt.json` 和 `选题研究报告.md`，写入 `知识点大纲.md`：

一、二、三等结构，共 4-7 个章节，每节含关键概念和注意事项。

### T2：口播稿

读取 `.step_prompt.json` 和 `知识点大纲.md`，写入 `配音稿_分段.txt`：

格式：--- P1 标题 (时长) ---\n内容...\n\n--- P2 ...

总 5-8 页，每页 100-300 字，口语化，无 ## ** `` 标记。

### T4：分镜方案 + image_slots

读取 `.step_prompt.json` 和 `配音稿_分段.txt`，写入两个文件：

- `PPT大纲.md`：每页布局分配
- `image_slots.json`：每页配图需求

### 通用规则

- 正文不使用 ##、**、``
- 口播稿无特殊字符
- 完成后删除 `.step_prompt.json`
