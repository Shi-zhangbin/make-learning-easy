# Narration Style · 口播话风技能

## 核心理念

**用最糙的话讲最硬的干货。**

把高深的技术概念翻译成人人都能秒懂的社会场景（渣男/丈母娘/买房/买菜/追女神）。不装、不端着、像在酒桌上跟朋友吹牛。

## 方法论

### Step 1 · 搜热点（写稿前置）

写任何内容前，先 WebSearch 搜索当前（年月）最新的网络热梗、热点事件、科技圈新闻。拿到至少 3-5 个能用的梗/热词/事件。**禁止用一个 2024 年的老梗**。

### Step 2 · 定叙事弧

全篇按这个结构：

```
开场抛问题（15秒钩子）→ 展开核心 → 中间名场面/转折 → 收束 → 结尾 callback 开头
```

### Step 3 · 每页塞一个类比

**每页至少一个** 社会场景类比，不能干讲技术概念。

| 概念 | 干讲 | 有梗的讲法 |
|------|------|-----------|
| LLM 知识截止 | LLM知识截止于训练数据 | **追女神类比**：你看过她全部微博，但不知道她中午吃了啥 |
| 向量搜索 | 计算余弦相似度找到最近邻居 | **川菜馆类比**：像在大众点评搜"附近好吃的川菜" |
| 模块化组件 | RAG各组件可独立升级 | **换显卡类比**：只换显卡不换CPU |
| RAG+微调 | 两者可组合使用 | **组合拳类比**：哪个拳击手只打直拳不打勾拳 |
| 上下文窗口限制 | 输入长度有限 | **塞不下类比**：就像你吃火锅点了 50 盘菜，桌子放不下 |

### Step 4 · 表达工具箱

轮换使用，不要重复：

| 类型 | 例句 |
|------|------|
| **横空断言** | "这事儿说白了就是…"、"你把它理解成…就行了" |
| **自黑自嘲** | "别急，我当初也是这么被忽悠的" |
| **荒诞类比** | "这就好比你去相亲…"、"像极了追你的女神突然…" |
| **装傻反问** | "是不是这个理儿？"、"你细品，是不是这个感觉" |
| **打断节奏** | "停，我知道你要问什么了。"、"等等，先别急着划走" |
| **弹幕诱饵** | "觉得有用的公屏扣1"、"懂的都懂" |

### Step 5 · 节奏控制

短-长-短呼吸节奏：

```
短断言 → 长解释 → 短总结/反问
```

例："RAG 是什么？说白了就四个字：先查后答。传统 LLM 是闭卷考试，你问它什么它只能凭记忆瞎编。RAG 相当于给了它一本可以随时翻的参考书——问之前先翻书，翻到再回答。你细品，是不是这个道理。"

### Step 6 · 结尾 callback

结尾必须呼应开头，形成弧线。不要用"本期视频就到这里"。

- 开头："为什么 AI 有时候像神、有时候像猪？"
- 结尾："所以回到开头那个问题——不是因为 AI 不行，是你没给它开卷的资格。"

## 校验规则

- 禁止传统开场白（"大家好"、"欢迎收看"、"本期视频"）
- 全篇至少出现 2 次 UP主口癖（"你细品"、"说白了"、"有一说一"等）
- 每页不超过 45 秒（约 200 字）
- 全篇要有"叙事弧"结构
- 不要引用具体 UP 主名称
