# Codex 操作指引

当 engine 输出 `📝 需要生成` 时，执行以下步骤：

1. 读取 `.step_prompt.json` 获取上下文
2. 参考此文档对应章节生成内容
3. 写入指定的输出文件
4. 删除 `.step_prompt.json`
5. engine 会自动继续推进

内容生成细节见 `skills/ascend-video-pipeline.md`

## 视频制作规则（必须遵守）

所有视频输出必须通过 Pipeline 在 episodes/ 目录下完成。
禁止的路径：
- ❌ 在项目根目录直接写 index.html
- ❌ 创建独立的 HyperFrames 项目（如 loop-engineering/）
- ❌ 绕过管线手动渲染

正确的做法：
1. python3 -m core.engine init "2026-06-27_主题_[Codex]" --topic "..." --style bilibili --tone talk-show
2. 写 episodes/2026-06-27_主题_[Codex]/timeline.json（元素数组格式）
3. bash go.sh run --episode "2026-06-27_主题_[Codex]" --step T6
